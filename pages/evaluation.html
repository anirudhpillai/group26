Critical Evaluation

Architecture Design

We are happy with the architecture design of our application. We tried to decouple everything as much as we so as to reduce dependencies and enable us to work on the project in parallel.
Our choice of frameworks and libraries (that had a huge influence on our architecture) was done following some guidelines:
How easy is it to get information and help (is there a wide community, ressources..) ? Can we build an application following important patterns (e.g.: decoupling our components, making unit tests easier, ect..) ?

User Interface design and user experience

We designed our app with the user in mind. We built our components to be as helpful and natural to use as possible. No fancy wheels or carousels etc.. We built elements with ease of use, understandability, and value proposition in mind. The user interactions are straightforward, with HCI principles in mind all along the way (For more information on how we focused on HCI throughout the app, visit this page ).
